## Video Walkthrough 
Twitter_API : A twitter mock application to acheieve network data transfers using RESTful API calls to manipulate data inside our app.

Tech : Utilize Twitter's RESTful API to GET, PUT, POST, and DELETE data. Network data transfers using iOS sdk objects such as URLsessionDataTask.
